**Install Anaconda Distribution (Prefer Python 3+)**

`Anaconda Linux`_

**Install Additional Python Modules**

Open Terminal and then enter:

.. code-block:: none

    conda install -c anaconda unixodbc
    conda install -c anaconda freetds

    pip install opedia
