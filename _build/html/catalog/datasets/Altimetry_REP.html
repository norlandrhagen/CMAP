Dataset Description¶

Simons CMAP contains Reprocessed Satellite Altimetry records from 2013-01-01 through 2018-06-10 collected from the Copernicus Marine Environment Monitoring Service. The dataset is generated from remotely sensed satellite data and contains multiple ocean altimetry products. The dataset has a spatial resolution of 1/4° X 1/4° and a daily temporal resolution.
